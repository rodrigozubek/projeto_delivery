# 🍺 Projeto Mobile - Delivery de Bebidas

Este projeto é uma aplicação de entrega de bebidas desenvolvida em Flutter para a disciplina de Desenvolvimento Móvel.

## 👥 Integrantes e Atividades
* **Denner:** : Atividades serão inseridas.
* **Renato:** Atividades serão inseridas
* **Rodrigo:**  Configuração inicial do projeto (`br.edu.utfpr.projetomobile`), estruturação das pastas (MVC), implementação do `BebidaController` (Provider) e criação da interface de catálogo. 

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Dart
* **Framework:** Flutter (Android)
* **Gerência de Estado:** Provider
* **Arquitetura:** MVC (Model-View-Controller)
